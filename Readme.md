# AI Expense Agent – Telegram Bot

## 📌 Description
AI Expense Agent is a Telegram-based AI bot that helps users track daily expenses using simple text messages.

## 🚀 Features
- Add expenses using natural language (example: add 50 groceries)
- Category-wise tracking
- Date-wise summary
- Telegram-based interface
- Cloud-hosted (24/7 availability)

## 🛠️ Technologies Used
- Python 3.11
- python-telegram-bot
- Telegram Bot API

## ▶️ How to Use
1. Open Telegram
2. Search for the bot
3. Use commands like:
   - add 50 groceries
   - /summary

## 🌐 Deployment
The bot is hosted on cloud and runs continuously without manual intervention.
